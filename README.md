# SistemaBarbearia
Gestão ágil para barbearias via dispositivos móveis
BarberControl - Sistema de Gerenciamento de Barbearia
Este projeto consiste em um sistema de automação para barbearias, desenvolvido em linguagem Python. O objetivo principal é oferecer uma ferramenta simples para o controle de atendimentos, permitindo gerenciar a entrada e saída de clientes, além de fornecer um resumo financeiro básico para o dia de trabalho.

Funcionalidades do Sistema
Cadastro de atendimentos: Registro do nome do cliente, serviço realizado e valor cobrado.

Gerenciamento de status: Permite acompanhar se o cliente está aguardando, em atendimento ou se o serviço já foi finalizado.

Relatório financeiro: O sistema percorre os registros finalizados e calcula o faturamento total e a comissão devida aos barbeiros.

Armazenamento de dados: Todas as informações são gravadas em um arquivo de texto (.txt), o que evita a perda de dados ao fechar o programa.

Estrutura de Arquivos
O código foi dividido em dois arquivos para facilitar a organização da lógica:

main.py: Contém o menu principal e a interação com o usuário.

funcoes.py: Contém os procedimentos de salvamento, carregamento de dados e cálculos.

Como Utilizar
Faça o download dos arquivos main.py e funcoes.py para uma pasta no seu computador ou celular.

Certifique-se de que o Python 3 está instalado.

Execute o arquivo main.py.

O arquivo data_store.txt será criado automaticamente na primeira vez que você salvar um registro.

Requisitos de Sistema
Python 3.x

Ambiente de terminal ou aplicativos como Pydroid 3 para execução em dispositivos móveis.
